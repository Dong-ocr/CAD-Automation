项目名称：
  CAD 自动化制图系统（Python + ezdxf）

项目简介（100-200字）：
  基于 Python + ezdxf 的 CAD 自动化出图系统，无需安装 AutoCAD 即可自动生成工程图纸。
  系统包含完整的绘图工具箱（cad_toolbox.py），支持线/圆/弧/标注/公差/螺纹/剖面线等
  工业制图标准元素，可一键输出 DXF/PDF/SVG/PLT 格式。已实现几何图形自动绘制、
  阶梯轴零件图生成、批量 CSV 驱动出图等功能，目标是覆盖 AutoCAD 常见作业需求，
  为没有 CAD 软件的用户提供自动化出图服务。

使用的 AI 工具：
  - Claude Code（日常编码助手）
  - Cursor（IDE 辅助开发）
  - MiMo API（计划接入，用于智能识别图纸需求并自动生成绘图脚本）

底层模型：
  - 当前：Claude（代码生成）
  - 计划：MiMo V2.5 系列（推理 + 多模态）

项目描述（详细）：
  我开发了一套完整的 CAD 自动化制图系统，核心是一个 18000+ 行的 Python 工具箱
  （cad_toolbox.py），封装了 ezdxf 的所有常用操作。系统支持：
  
  1. 基本图元：直线、圆、圆弧、矩形、多段线、文字
  2. 高级标注：水平/垂直尺寸、表面粗糙度、形位公差框格
  3. 专业功能：外螺纹表示、螺纹牙放大、剖面线填充、圆阵列
  4. 输出格式：DXF / PDF / SVG / PLT（HPGL）
  5. 图框系统：A3/A2/A1 带标题栏自动填充
  6. 批量处理：CSV 驱动批量生成零件图
  
  计划将 MiMo 的推理能力接入系统，实现"自然语言描述→自动生成绘图脚本"的
  端到端流程，大幅降低工程制图的门槛。
