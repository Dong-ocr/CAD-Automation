# SL 73 水利水电工程制图标准 参考

## 适用范围
SL 73-2013 水利水电工程制图标准，适用于水电站、大坝、渠道、泵站等水利工程图纸。

## 水工常用符号
- 水位符号: 波浪线 + WL + 高程值
- 闸门: 矩形 + 对角线交叉填充
- 管道: 虚线 + 管径标注
- 泵站: 圆形 + P 标记
- 阀门: 三角形或圆形交叉
- 消力池: 阶梯形轮廓

## 图幅标准
- 水电站施工图常用 A1(594×841) 或 A0(841×1189)
- 比例: 平面 1:100~1:500, 剖面 1:50~1:200
- 高程基准: 黄海高程系

## 标注规则
- 尺寸标注: mm 为单位，不写单位
- 标高: m 为单位，精确到 0.01m
- 坡度: 1:m 或 % 表示
- 坐标: 桩号 + 偏移量

## 钢筋表示
- 一级钢 HPB300: 圆点
- 二级钢 HRB335: 单斜线
- 三级钢 HRB400: 双斜线
- 间距: @ 表示

## 水电站特有图纸类型
1. 枢纽布置图
2. 挡水坝段图
3. 厂房横剖面图
4. 压力钢管布置图
5. 机电安装图
6. 金属结构图
7. 施工导流图

## 出图流程
1. 在 Model Space 以 1:1 比例绘制
2. 在 Paper Space 设置视口比例
3. 使用 CTB 打印样式控制线宽
4. 标准图框 + 会签栏 + 图例
