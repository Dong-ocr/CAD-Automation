1. 项目解决的核心痛点：

  传统 CAD 制图依赖昂贵的 AutoCAD 正版软件（年费数千元）和大量手动操作，
  对于学生、自由职业者和小型团队来说门槛极高。具体痛点包括：

  (a) 软件成本高：AutoCAD 正版年费对学生群体不友好
  (b) 重复劳动多：工程制图中大量重复性工作
  (c) 技能门槛高：非机械专业的人需要临时学 CAD
  (d) 格式兼容差：不同 CAD 版本之间的格式转换困难

2. 核心逻辑流（长链推理 + 多工具协作）：

  【第一层：需求理解层】
  - 通过 Claude Code（当前）/ MiMo（计划接入）理解用户描述
  - 将自然语言需求解析为结构化参数

  【第二层：代码生成层】— 长链推理核心
  - AI 根据需求参数调用 cad_toolbox.py 工具箱 API
  - 涉及复杂的几何推理：圆弧相切计算、对称键槽定位、
    螺纹牙三角形阵列生成、斜二测投影坐标变换
  - 每个图形元素需要精确的数学计算（三角函数、坐标系变换）
  - 形成 10-20 步的推理链，一步出错则整张图纸偏移

  【第三层：渲染输出层】
  - ezdxf 引擎生成标准 DXF 文件
  - matplotlib 渲染 PDF 预览
  - HPGL 转换器生成 PLT 文件
  - 多格式输出，无需用户安装任何 CAD 软件

  【第四层：批量自动化层】
  - CSV 驱动 + 循环遍历，实现填表即出图
  - 支持一次处理数百个零件尺寸

  实际案例推理链（阶梯轴零件图）：
  用户输入 -> AI 解析为 5 段轴数据
  -> 遍历每段绘制外轮廓（推理轴肩相邻关系）
  -> 计算键槽位置（居中偏移）
  -> 构建剖面线边界多边形（处理键槽缺口）
  -> 计算斜二测投影坐标（45度角变换）
  -> 生成螺纹局部放大图（3x 缩放）
  -> 自动分布尺寸标注避免重叠
  -> 输出 DXF+PDF
  整个推理链约 30+ 步连续几何计算，属于典型长链推理场景。

3. 具体成果量化：

  - 代码量：工具箱 18000+ 行 Python，封装 50+ 个绘图 API
  - 测试覆盖率：24 项自动化测试全部通过
  - 出图速度：从参数到最终 PDF < 2 秒
  - 已实现：几何作图 / 阶梯轴 / 校园平面 / 圆弧连接 / 对称图形
  - 输出格式：DXF + PDF + SVG + PLT 四格式一键导出
